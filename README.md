# ShopWaveLite API

A simple e-commerce REST API built with ASP.NET Core (.NET 10), PostgreSQL (Neon), and JWT authentication.

## Tech Stack
- ASP.NET Core Web API (.NET 10)
- Entity Framework Core (PostgreSQL)
- JWT + Refresh Tokens
- Roles: Admin, Vendor, Customer
- Hosted on Render

## Features
- User auth (register, login, refresh token)
- Product management
- Order management
